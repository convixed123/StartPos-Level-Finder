# StartPos Level Finder

Adds a new button to quickly find and play startpos version of any level!

## Features:

* "Start Pos" button in front of the "View Level" button
* Start Pos search layer for easier search
* Search bar for long or hard-to-find level names(ex. "Super Probably Level")
* Star button which represents a "pin" button to pin your sp level
* Dark aqua button in level page to unpin the level

## How to use:

* Press "Start Pos" button to open up level search
* Next, find a sp level you like the most and press tiny star(pin) button
* "Start Pos" button will become green
* Press it and see the *Magic*
